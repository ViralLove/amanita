Интеграция WooCommerce с блокчейн экосистемой AMANITA через Python микросервис.

== Description ==

Amanita Connector - это мост между привычной e-commerce средой WooCommerce и децентрализованной экосистемой AMANITA. Плагин обеспечивает автоматическую синхронизацию товаров и заказов с блокчейном, управление инвайтами и аналитику.

= Основные возможности =

* **Интеграция с блокчейном** - автоматическая синхронизация товаров и заказов
* **Управление товарами** - метабокс с информацией о синхронизации, статус в реальном времени
* **Система инвайтов** - гибкие правила выдачи инвайтов покупателям
* **Безопасность** - HMAC подписи, rate limiting, безопасное хранение ключей
* **Логирование** - подробные логи всех операций

= Архитектура =

```
[WooCommerce Store] ↔ [Amanita WP Plugin] ↔ [Python Microservice] ↔ [Blockchain]
```

= Синхронизируемые данные =

* Основная информация товара (название, описание, цена)
* Категории и теги
* Изображения продукта
* Атрибуты и вариации
* Информация о наличии
* Детали заказов
* Информация о клиентах

= Требования =

* WordPress 5.0+
* WooCommerce 3.0+
* PHP 7.4+
* MySQL 5.7+

== Installation ==

1. Скачайте архив плагина
2. В админке WordPress перейдите в **Плагины → Добавить новый**
3. Нажмите **Загрузить плагин** и выберите архив
4. Активируйте плагин
5. Перейдите в **WooCommerce → Amanita** для настройки

== Frequently Asked Questions ==

= Как настроить интеграцию? =

1. Перейдите в **WooCommerce → Amanita**
2. Введите URL вашего Python микросервиса
3. Настройте параметры блокчейна и IPFS
4. Протестируйте соединение

= Какие данные синхронизируются? =

Плагин синхронизирует всю основную информацию о товарах: название, описание, цену, категории, изображения, атрибуты, вариации и информацию о наличии.

= Как работает автоматическая синхронизация? =

При включенной автоматической синхронизации товары и заказы будут синхронизироваться автоматически при создании или обновлении через Action Scheduler.

= Безопасно ли хранить ключи в плагине? =

Да, плагин использует безопасные методы хранения ключей WordPress и не передает их в открытом виде.

= Поддерживается ли мультисайт? =

Да, плагин полностью совместим с WordPress Multisite.
